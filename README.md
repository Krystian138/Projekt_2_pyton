# Projekt 2 – Mini SCADA (Python / PyQt5)

Prosta aplikacja typu „mini-SCADA” pokazująca instalację z 4 zbiornikami (T1–T4) oraz elementami sterowania:
- zawór dopływu do T1 (0–100%)
- grzałka w T1 (0–100%)
- pompa na obejściu/recyrkulacji (przepływ w L/min)
- przelewy grawitacyjne po przekroczeniu poziomu króćców
- odpływ z dołu zbiornika T4
- alarmy (np. za niski / za wysoki poziom, itp.)
- dodatkowy widok „instalacja/wykresy” 

## Wymagania
- Python 3.10+ 
- PyQt5

Instalacja PyQt5:

pip install PyQt5
